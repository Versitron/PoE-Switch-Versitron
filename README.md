## Frequently Asked Questions

### What is a PoE+ switch?  
A PoE+ switch is a network switch that provides both Gigabit Ethernet connectivity and enhanced Power over Ethernet (up to 30W per port) to connected devices.

### What devices benefit from PoE+?  
Devices such as high-resolution IP cameras, wireless access points, VoIP phones, and other PoE-enabled equipment requiring higher power than standard PoE.

### Are PoE+ switches compatible with standard PoE devices?  
Yes PoE+ switches are backward compatible with devices using IEEE 802.3af (standard PoE).
